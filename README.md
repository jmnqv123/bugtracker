# bugtracker
This code defines a Bug class that represents a single bug, with three attributes: summary, description, and priority. It also defines a BugTracker class that has a list of Bug objects and provides several methods for managing the list of bugs, including add_bug, list_bugs, and find_bug.

The BugTracker class can be used to create a bug tracker as follows:

Create an instance of the BugTracker class.
Add some bugs to the tracker using the add_bug method.
List all the bugs using the list_bugs method.
Find a specific bug using the find_bug method.
